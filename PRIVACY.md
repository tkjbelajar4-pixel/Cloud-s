# Kebijakan Privasi Neverlabs Cloud's

Terakhir diperbarui: 22 Februari 2026

Terima kasih telah menggunakan Neverlabs Cloud's kami. Privasi Anda adalah hal yang penting bagi kami. Halaman ini menjelaskan bagaimana aplikasi ini menangani data Anda.

## 1. Data yang Dikumpulkan

Aplikasi ini **TIDAK mengumpulkan data pribadi apa pun** ke server eksternal. Semua data yang Anda unggah atau buat (termasuk file, folder, catatan teks) disimpan secara lokal di perangkat Anda menggunakan teknologi **IndexedDB** dan **localStorage** milik browser. Data tersebut tidak pernah dikirim ke server kami atau pihak ketiga mana pun.

## 2. Penggunaan Data Pihak Ketiga

Aplikasi ini menggunakan layanan pihak ketiga berikut:

- **Font Awesome** – untuk ikon. Font Awesome dapat mengumpulkan data penggunaan sesuai kebijakan privasi mereka.
- **Google Fonts** – untuk tampilan font. Google mungkin mengumpulkan data sesuai kebijakan privasi mereka.

Kami tidak mengirimkan data pribadi Anda ke layanan tersebut.

## 3. Cookie dan Teknologi Pelacakan

Aplikasi ini tidak menggunakan cookie untuk pelacakan. Kami hanya menggunakan **sessionStorage** untuk menyimpan sementara folder aktif saat ini, dan **localStorage** untuk konfigurasi tampilan (seperti preferensi folder). Semua ini bersifat lokal dan tidak dilacak.

## 4. Keamanan Data

Karena data Anda hanya disimpan di perangkat Anda sendiri, keamanan data bergantung pada keamanan perangkat dan browser Anda. Kami sarankan untuk:

- Tidak membagikan akses fisik ke perangkat Anda.
- Menggunakan browser yang selalu diperbarui.
- Melakukan pencadangan (ekspor) data secara berkala melalui fitur yang tersedia didalam aplikasi.

## 5. Hak Pengguna

Anda memiliki kendali penuh atas data Anda. Anda dapat:

- Menghapus file atau folder kapan saja.
- Menghapus seluruh data melalui tombol "Hapus Semua Data" di menu Settings.
- Mengekspor data dalam format JSON sebagai cadangan.

## 6. Perubahan Kebijakan Privasi

Kebijakan privasi ini dapat diperbarui dari waktu ke waktu. Setiap perubahan akan diumumkan melalui pembaruan aplikasi. Dengan terus menggunakan aplikasi ini, Anda dianggap menyetujui kebijakan privasi terbaru.
