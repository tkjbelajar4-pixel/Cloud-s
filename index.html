Konfigurasi Cloudinary

Masukkan Cloud Name dan Upload Preset dari akun Cloudinary Anda. Data ini disimpan hanya di perangkat Anda.

Cara mendapatkannya:
1. Daftar/login di Cloudinary
2. Salin Cloud Name dari dashboard.
3. Buat Upload Preset (Settings > Upload > Add upload preset, pilih unsigned).
4. Masukkan kedua nilai di bawah.
